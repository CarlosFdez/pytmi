Welcome to pytmi’s documentation!¶

PyTMI is a python library for communicating in the Twitch Messaging Interface (docs).

This library supports core communication via the pytmi.TwitchClient class, and bot command parsing via the pytmi.bot.TwitchBot subclass. Understanding asyncio is crucial to using this library, as bot events and commands are implemented as coroutines. This allows concurrency without the need for multiple threads.
